# LuxeCalendar

A premium, interactive React-based physical wall calendar component built with Next.js, Framer Motion, and Tailwind CSS.

## Features
- **Modern Physical Aesthetic**: Two-fold "paper" layout with spiral binding and high-quality imagery.
- **Range Selection**: Intuitive click-to-select range logic with hover feedback.
- **Page Flip Animations**: 3D vertical month transitions.
- **Persistent Memos**: Monthly notes saved to `localStorage`.
- **Responsive**: Seamless transitions between desktop spatial layout and mobile scroll layout.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Date Logic**: date-fns
- **Icons**: Lucide React

## Getting Started

1.  Navigate to the `LuxeCalendar` directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
